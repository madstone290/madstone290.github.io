# Kramdown Utility 

## Kramdown 클래스
{: #아이디 .클래스1 .클래스2 } 텍스트를 추가함으로써 html 변환시 id 및 class 속성을 추가할 수 있다.

마크다운
```
Left aligned text
{: .text-left}
```
HTML
```
<p class="text-left">Left aligned text</p>
```
